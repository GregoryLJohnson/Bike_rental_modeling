# Bike_rental_modeling
A machine learning tutorial I made for a NCSU physics grad student machine learning reading group.

This was the result of a tutorial session I put together for a machine learning reading group which consisted mainly of physics graduate students at NCSU. My motivation was to illuminate the statistical modeling aspects of machine learning such as the relationship between loss functions and implied likelihoods, modeling techniques for simple non-linear features, and the import of a neural network for modeling an aribtrary, possibly non-linear fit function instead of a handmade fit.

The pdf gives an overview of the concepts involved in the actual exploration/modeling of the bike rental dataset. The EDA file is a exploratory data analysis. The Modeling file contains the evaluation of various models and a discussion of their performance.
